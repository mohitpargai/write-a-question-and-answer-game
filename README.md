# write-a-question-and-answer-game
Built a quiz-based Question and Answer Game with score tracking, answer validation, and an interactive user interface to provide an engaging learning experience.
# ज्ञान मंच - Hindi Q&A Game

यह एक static Hindi question-answer game है. इसे किसी backend या database की जरूरत नहीं है.

## Local Open

`index.html` file को browser में open करें.

## Global/Public Link कैसे बनाएं

### Option 1: Netlify

1. https://app.netlify.com/drop खोलें.
2. इस पूरे folder को drag and drop करें:
   `write-a-question-and-answer-game`
3. Netlify आपको public URL दे देगा.

### Option 2: GitHub Pages

1. GitHub पर एक नया repository बनाएं.
2. इस folder की files upload करें:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
3. Repository settings में जाएं.
4. Pages section में branch को `main` और folder को `/root` select करें.
5. Save करने के बाद GitHub public URL देगा.

## Files

- `index.html`: game layout
- `styles.css`: design
- `app.js`: Hindi questions, answers, timer, lifelines, music
